Get Fist Contentful Paint (FCP)

The First Contentful Paint (FCP) metric measures the time from when the page starts loading to when any part of the page's content is rendered on the screen.

This event emit as soon as the FCP is available. This event will only be emitted once.

When this event is emitted, the context data will be appended to the event object as described in the API. The values will be available in a special property on the event object 

event.$webVitals
