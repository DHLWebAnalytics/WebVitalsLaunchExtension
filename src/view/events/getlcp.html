Get Largest Contentful Paint (LCP)

Largest Contentful Paint (LCP): measures loading performance. To provide a good user experience, LCP should occur within 2.5 seconds of when the page first starts loading.

This event will emit whenever a LCP score is calculated. The function will be called any time a new largest-contentful-paint performance entry is dispatched, or once the final value of the metric has been determined. You can change it to emit every time a CLS is calculated by ticking the checkbox.

When this event is emitted, the context data will be appended to the event object as described in the API. The values will be available in a special property on the event object 

event.$webVitals
